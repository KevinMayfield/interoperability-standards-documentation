Arranging a proxy relationship



  1. The national proxy request is captured in a form which is based on Proxy Access Request, this is submitted to the National Proxy Service (example completed national proxy request form)
  2. The National Proxy Service uses this form to create FHIR Consent resource which follows National Proxy FHIR Consent profile, example proposed Consent

  3. The National Proxy Service reviews this consent

    a. If further verification is required. The patients GP will be asked to verify the Consent via a Task, example Task national proxy request verification

  4. ​The consent is now either active Example Consent active or rejected Example Consent rejected
